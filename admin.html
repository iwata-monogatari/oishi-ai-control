GitHubへ正式反映する手順
  1. この画面でAIを登録・編集する。
  2. 「JSONをコピー」を押す。
  3. GitHubで data/current.json を開く。
  4. 鉛筆マーク、または Edit this file を押す。
  5. 中身を全部消して、コピーしたJSONを貼る。
  6. Commit changes を押す。
  7. 管制塔画面に戻って再読み込みする。
